# Wedding Invitation

Одностраничный сайт-приглашение на свадьбу на React + TypeScript.

## Совместимость

Проект зафиксирован под **Node.js 12.22+** и использует версии пакетов, совместимые с Node 12.

Если до этого в проекте уже стояли более новые версии Vite, обязательно переустановите зависимости с нуля:

```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

## Что внутри

- элегантный hero-блок с таймером до даты свадьбы;
- секции с деталями, программой и дресс-кодом;
- форма RSVP с клиентским подтверждением отправки.

## Сборка

```bash
npm run build
```

## Кастомизация

Основной контент вынесен в `src/App.tsx`:

- дата свадьбы: `weddingDate`;
- программа: `program`;
- блок деталей: `details`;
- палитра дресс-кода: `palette`.
