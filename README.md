# Wedding Invitation

Одностраничный сайт-приглашение на свадьбу на React + TypeScript.

## Что внутри

- элегантный hero-блок с таймером до даты свадьбы;
- секции с деталями, программой и дресс-кодом;
- форма RSVP с клиентским подтверждением отправки.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Кастомизация

Основной контент вынесен в `src/App.tsx`:

- дата свадьбы: `weddingDate`;
- программа: `program`;
- блок деталей: `details`;
- палитра дресс-кода: `palette`.
